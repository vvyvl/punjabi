## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A Punjabi music file (MP3 format)

### Setup Instructions

1. **Clone or download** this repository to your local machine
2. **Add your Punjabi song**:
   - Place your `.mp3` file in the root directory
   - Name it `punjabi-song.mp3` (or update the filename in `index.html` line 12)
3. **Open `index.html`** in your web browser
4. **Enjoy!** Music will auto-play when the page loads

### Adding Your Own Music

To use a different Punjabi song:
1. Open `index.html`
2. Find line 12: `<source src="punjabi-song.mp3" type="audio/mpeg">`
3. Replace `punjabi-song.mp3` with your audio file name
4. Save and refresh the page

## 🛠️ Customization

### Change Colors
Edit the color values in `style.css`:
```css
.header h1 { color: #d4641b; } /* Change heading colors */
.btn { background: linear-gradient(135deg, #ff6b35, #f7931e); } /* Change button colors */
```

### Modify Content
Edit `index.html` to:
- Change samosa recipe ingredients
- Add your own stories about samosas
- Update contact information
- Add more images to the gallery

### Update Music
In `index.html`, modify line 12:
```html
<source src="your-music-file.mp3" type="audio/mpeg">
```
